# e-plantShopping

e-plantShopping is a React-based plant shopping web application where users can browse different categories of plants and add them to their shopping cart.

## Project Overview

The application demonstrates the implementation of a modern front-end e-commerce interface using React and Redux.

Users can:

- Browse plants by category
- View plant details including name, price, and image
- Add plants to the shopping cart
- Update quantities in the cart
- Remove items from the cart
- View the total cart amount
- Proceed to checkout (Coming Soon)

## Technologies Used

- React
- Redux Toolkit
- JavaScript
- CSS

## Features

- Product listing page with plant categories
- Shopping cart with quantity management
- Dynamic cart total calculation
- Navigation between pages
